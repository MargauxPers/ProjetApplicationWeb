## Projet d'application Web 

Ce projet correspond à un développement réalisé dans le cadre de l'enseignement INF-TC3 en première année de l'Ecole Centrale de Lyon

Choix des informations utilisées : 
 - Nom
 - Capitale
 - Longitude 
 - Latitude
 - Densité
 - Superficie
 - Leader du pays 
 - Langues 
 - Monnaie 
 - IDH
 - Moyenne d'âge
